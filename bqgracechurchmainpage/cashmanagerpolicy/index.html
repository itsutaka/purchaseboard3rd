第一條：零用金之建立與保管人指定

本條款規範了零用金的申請設立流程、保管人如何確認，以及首次撥款和安全保管的具體要求，確保零用金從源頭開始便有妥善管理。

1. 申請與核准: 由需求同工提出申請，詳述需要指定人員保管零用金之理由，例如：支應社團法人主恩改革宗長老會（以下簡稱本社）會務活動、行政庶務或現場臨時性現金支付需求，建議保管人選、以及預計保管之零用金定額（建議根據過往經驗或活動預算，設定足以應付數次小額支付，但不過高之固定金額，例如新台幣 NT$10,000 ~ NT$50,000）。申請案需依本社內部分層負責授權規定，經秘書長及財務人員（或財務委員會）審核，並呈報【理事長/常務理事/依本社章程或理事會決議之權責人員】核准後始得設立。若預期會有現金需求大於固定金額，則歸類為大額支出，也依照此流程於付款前提早另立一案逕行申請。

2. 保管人確認: 經正式指定之保管人，需簽署本社「零用金保管責任書」，以確認其已充分瞭解並同意遵守本管理辦法及相關規定。

3. 首次撥款: 本社財務人員（或出納）應依核准之定額，將首次現金零用金撥付予保管人，保管人需簽收確認。財務人員應詳實記錄此筆撥款。

4. 安全保管: 保管人必須將本零用金獨立存放於本社指定之上鎖容器（如：現金箱、帶鎖抽屜或保險櫃）中，鑰匙由保管人專責妥善保管，非經授權不得將現金或鑰匙交付他人。嚴禁將本零用金與私人款項或其他非本社款項混合存放或挪用。本零用金僅能用於支付經本社核准之公務相關支出。

第二條：支付作業

本條款詳細說明了零用金的適用範圍、單筆支付上限，以及在支付過程中如何取得合法憑證和進行支付記錄，旨在規範零用金的合規使用。

1. 適用範圍: 本零用金明確規定僅能用於「事先約定或臨時發生，且收款廠商/個人明確要求現金支付之小額款項或事先約定之大額支付款項，無法採用轉帳、劃撥方式進行付款」之情形（例如：會務活動所需之臨時採買、郵資、交通費、講座講師臨時雜支、小型維修工資、運費、臨時少量庶務用品等）。

2. 單筆支付上限設定: 本社設定零用金單筆支付上限為新台幣 NT$6,000元整。若遇特殊情況確需動用本零用金支付超過單筆上限之款項，必須事先取得理事會同意。

3. 取得合法憑證: 支付任何款項時, 保管人必須向收款方索取符合臺灣稅法及本社財務規定之合法憑證（如：統一發票、免用統一發票收據、或其他經認可之正式收據）。憑證上需清晰載明：收款單位/個人之名稱及統一編號（或身分證號）、交易日期、消費內容（品名、規格、數量、單價）、總金額（含稅，並註明大小寫），且應加蓋收款單位之統一發票專用章或正式店章/負責人章。若廠商僅能提供普通收據，務必確認上述資訊齊全並有廠商蓋章。若遇極少數情況（如偏遠地區小販、農產品交易等）確實無法取得前述憑證，應於「零用金支付記錄」上詳述理由，並盡可能取得第三方人證或拍照存證，且需依本社規定經理事會特別核准後方可報支。

4. 支付記錄: 保管人需使用本社規定之「零用金支付記錄」（可為紙本簿冊或指定之電子表單/App）。每一筆支付，不論金額大小，都必須立即、詳實記錄以下資訊：

  • 支付日期 (Date)
  • 請款人 (Payee A)
  • 收款廠商/對象 (Payee B)
  • 會計科目 (Description/Purpose)
  • 支付金額 (Amount Paid)
  • 記錄後現金餘額 (Remaining Balance)
  • 保管人簽名/註記 (Payer's Signature/Note)

第三條：核銷與補款

本條款闡述了零用金核銷的時機、準備核銷文件所需的項目與核對公式、詳細的審核流程，以及最終的補款作業，確保零用金帳目清晰並及時補充。

1. 核銷時機: 當本零用金餘額降低至預設之最低水位（例如：原定額之20%），或依本社規定之固定週期（例如：每週末、每月初、或每次大型活動結束後）必須辦理核銷。

2. 準備核銷文件: 保管人應將核銷期間內所有支付之原始憑證（發票、收據等）按「零用金支付記錄」之順序整理、編號並黏貼（若為紙本）。填寫本社標準格式之「零用金核銷暨補款申請單」，內容應包含：保管人姓名、職稱/會員編號、核銷期間、清點日期、期初零用金定額、本次核銷憑證總金額、盤點實存現金餘額、帳面應有現金餘額、差異金額（若有，需說明原因）、以及申請補款金額。核對公式：期初定額 - 本次核銷總額 = 帳面應有現金餘額（必須與實存現金 + 憑證總額相符）。

3. 審核流程: 保管人將「零用金核銷暨補款申請單」連同所有原始憑證及「零用金支付記錄」（影本或正本供查驗）提交給本社規定之初審人員審核簽署。初審人員確認無誤後，轉交本社財務人員（或財務委員會/指定財務審核人員）進行專業複核。財務人員需確認：憑證之合法性與合規性、金額計算之正確性、支出與記錄是否相符、是否符合本社章程及相關財務規定。

4. 補款作業: 經財務人員（或財務委員會/指定財務審核人員）審核無誤後，依「零用金核銷暨補款申請單」上核准之補款金額，由本社財務人員（或出納）將現金撥付給保管人，使其手上持有之現金恢復至原零用金定額。保管人點收補款現金無誤後應簽收。財務人員應將已核銷之各項費用登載入帳。

第四條：管理與稽核

本條款強調保管人的日常職責、定期與不定期的抽查機制、保管人異動時的處理程序，以及辦法的宣導與修訂要求，旨在建立完善的監督與持續改進機制。

1. 保管人職責: 保管人需隨時確保「手上庫存現金」加上「已支付但尚未核銷之有效憑證總額」永遠等於「零用金定額」。

2. 定期/不定期抽查: 本社財務人員、內部稽核人員（若有設置）或監事（會）指派之人員，應定期（例如：每月）及不定期（突襲檢查）對保管人所保管之零用金進行盤點，確認帳實（現金 + 單據）是否相符，並檢視支付記錄與憑證是否合規。盤點及抽查結果應作成書面記錄，如有異常應立即通報相關權責主管及監事（會）。

3. 保管人異動: 若保管人職務異動、離職或離任，必須依本社規定辦理零用金之結清、盤點與移交（或繳回）程序。此程序應由財務人員及或監事會指派之監交人共同辦理，確認無誤後解除原保管人之責任。

4. 辦法宣導與修訂: 本管理辦法應讓所有保管人、相關業務承辦人、理監事充分了解。本辦法應依本社章程、相關法規及實際運作需要，由財務組至少每年檢討一次，並視需要提報理事會（或會員大會，依章程規定）審議修訂。

第五條：施行要點

本條款列出了確保本辦法有效施行的關鍵措施，包括對相關人員的教育訓練、所需表單的標準化、各級簽核人員責任的落實，以及對未盡事宜的處理原則。

1. 教育訓練: 對新任保管人及相關人員進行清楚之教育訓練，確保其了解本辦法所有流程、表單填寫及應負責任。

2. 表單標準化: 設計本社標準化之「零用金支付記錄」（或規範電子記錄系統之操作標準）、「零用金核銷暨補款申請單」及「零用金保管責任書」。

3. 簽核權責落實: 強調各級簽核人員（含秘書長、財務人員、理事、監事等）於相關文件上簽署之責任，該簽署代表已就其權責範圍內確認支出之合理性、必要性及合規性。

4. 遵守法規: 本辦法未盡事宜，悉依本社章程、相關財務處理規定及中華民國相關法令規章辦理。
